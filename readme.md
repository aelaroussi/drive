# BYTECACHE

Public Google Drive folder interface. [bytecache.vercel.app]

## Tech

HTML, CSS, JS. Google Drive API v3.

## Features

- Dynamic file listing.
- MIME-type icon mapping.
- Direct download links.

## Config

Edit `main.js`:

```javascript
const API_KEY = "YOUR_API_KEY";
const FOLDER_ID = "YOUR_FOLDER_ID";
```

Target folder must be set to "Anyone with the link can view".
